About


UnBBayes (Matsumoto et al., 2011) is a probabilistic network framework written in Java. It has both a GUI and an API with inference, sampling, learning and evaluation. It supports Bayesian networks, influence diagrams, MSBN, OOBN, HBN, MEBN/PR-OWL, PRM, structure, parameter and incremental learning. Please visit the UnBBayes official website for more information.

Human-aided Multi-Entity Bayesian Networks Learning (HML) (Park et al., 2016) is a framework to develop a MEBN model from the domain expert's knowledge combined with relational data.

The HML-UnBBayes Plugin is an adapter between UnBBayes-MEBN and HML core.
