HML-UnBBayes Plugin

The HML (Human-aided Multi-Entity Bayesian Networks Learning) tool is now incorporated into UnBBayes as a brand new plugin! Adding this plugin will enable users to create Multi-Entity Bayesian Networks (MEBN) from a relational database (RDB), learning Local Probability Distributions (LPD) of nodes from the data on the RDB.

Start using it now!
