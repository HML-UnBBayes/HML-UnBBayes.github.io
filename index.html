HMLP Plugin

The HMLP (Human-aided Multi-Entity Bayesian Networks Learning Process) tool is now incorporated into UnBBayes as a brand new plugin! Adding this plugin will enable users to create Multi-Entity Bayesian Networks (MEBN) from a relational database (RDB), learning Local Probability Distributions (LPD) of nodes from the data on the RDB.

Start using it now!

New UnBBayes plugins will be added to this website in future! Feel free to bookmark it!
